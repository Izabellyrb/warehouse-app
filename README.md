# warehouse-app

Aplicação que gerencia galpões e o estoque de cada galpão (projeto de estudo do TreinaDev 9).
<br /><br />
### Versão do Ruby/Rails
- ruby 3.0.4 
- Rails 7.0.4
<br /><br />
### Gems necessárias
- rspec-rails 5.1.2 
- capybara 3.37.1 
- devise 4.8.1
- bootstrap 5.2.1
